Time to get stylish


Fun stuff! Get excited!


Today we're going to manipulate styles through JavaScript. This will allow you to:

  • change colors
  • change sizes
  • show and hide things on a page
  • dynamically change any CSS properties you want

There are two ways to change the style of an element:

  • Change the style directly
  • Change the class name

Style an element directly

Type out the following example and open it in your browser to see how it works:


				<div id="mydiv">Example</div>

				<script>
				document.getElementById('mydiv').style.color = '#ff0000';
				document.getElementById('mydiv').style.backgroundColor = 'blue';
				</script>

Horrible color scheme aside, you can start to see a pattern here:

It's basically [element] dot [style] dot [css property] equals [value].

But how does it actually work?

The direct way you just learned to change styles through JavaScript works by changing the style property of the HTML element.

In other words, this script:


					document.getElementById('mydiv').style.color = 'green';

...would change something like this:


					<div id="mydiv">My div</div>

...into something like this:


					<div id="mydiv" style="color:green;">My div</div>

How to hide and show HTML elements with JavaScript

Hide and show elements by changing the CSS display property to "none" or "block" (or "inline-block" if needed).

Here's an example that hides a div. (Type it to see it in action.)


				<div id="mydiv">You can't see me.</div>

				<script>
				document.getElementById('mydiv').style.display = 'none';
				</script>

Warning about CSS property names

For normal CSS properties like color or padding, it's all lowercase, but for hyphenated CSS properties like background-color or padding-top, you need to write them like this in JavaScript: backgroundColor or paddingTop.

That's because JavaScript thinks the hyphen is a minus sign (unless it's inside a string) and the browser would get really confused really fast.

Change an element's class name

Type out the following example and open it in your browser to see how it works:


				<style>
				.highlighted{
					background-color: yellow;
				}
				</style>

				<span id="mytext">Here's some text.</span>

				<script>
				document.getElementById('mytext').className = 'highlighted';
				</script>

Warning about CSS class names in JavaScript

Remember to use className if you want to change an element's class through JavaScript.
