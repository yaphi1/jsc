String methods: Get better control of your text

JavaScript allows you to do many things with your text, but here are the two we're going to look at in this section:

  • String slice: Get a portion of text from a string
  • String replace: Replace some text

String slice:
Get a portion of text from a string

The string slice method lets you grab a slice of text from a string.

Try the following example to see how it works:


				var str = "Here is some text.";
				var result = str.slice(1,7);

				console.log(result);

The result should say "ere is".

(Remember, 0 is the first character position, so 1 is the second.)


Choosing the start and end position of a string slice

The general format of slice is:


				slice(startPosition, endPosition)

But the end is optional.

If you want, you can just use slice(start) and it will give you everything from the starting position until the end of the whole string. Try the example:


				var str = "Here is some text.";
				var result = str.slice(1);

				// result should say "ere is some text."
				console.log(result);

Want to get the last few characters of a string?

Use a negative number.

slice(-3) would get the last 3 characters.

Try the following example to see how it works:


				var str = "Here is some text.";

				// last 5 characters
				var result = str.slice(-5);

				console.log(result);

In that example, we used a negative number as the start position. In other words, we told our script to start at the fifth-to-last character.

You can use a negative number as the end position too.

Let's say we want to get all but the first two and last two characters of a string. Try this example to see how that would work.


				var str = "xxHere is some text.xx";
				var result = str.slice(2,-2);

				console.log(result);

String replace: Replace parts of your text.

Try the following example. If you've done it correctly, it will find the first instance of "some" and replace it with "more".


				var str = "Here is some text.";
				var result = str.replace('some', 'more');

				console.log(result);

The result should say "Here is more text."

What if you want a more advanced version of replace? What if you want to replace multiple things at once? What if you want to replace things that follow a general pattern?

Quick recap of slice and replace

slice grabs a slice of text like so:

myString.slice(start, end)

replace replaces text like so:

myString.replace('old text', 'new text')
