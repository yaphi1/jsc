Basic functions: Super important in JavaScript

In JavaScript, a function is a piece of code that you can write once and run it as many times as you want in other places.

Type the following code into your script and open it in your browser:


				// first you define the function
				function sayStuff(){

					// all the code in here will run whenever you call your function
					console.log('hi');
					console.log('this is neat');
				}

				// now you can run the function whenever you want
				sayStuff();

				// run the function again just for fun
				sayStuff();

Play around with the code to get comfortable with how it works.

Return statements: Good for flexibility

A return statement stops a function and spits out a value.

This concept is easier to see with an example (which you should type out and try once):


				function add(){
					var sum = 1 + 5;
					return sum;
				}

				console.log( add() );

				console.log( 'The sum is: ' + add() );

In the above example, you can call add() and it will give you the sum of 1 and 5. You can use this value anywhere you want.

For example, you can log it to the console, or you can display it on your page.

Another bonus feature of return statements is that anything after them inside a function won't run. This is useful if you ever want to stop a function early.

Parameters: Make your functions more powerful and convenient

A parameter is a value you can plug into a function.

For example, you could write a function that takes in a name and outputs a greeting.

Type the following example and run it in your browser to see how it works:


				function greet(name){
					console.log( 'Hi ' + name );
				}

				greet('Mr. Meow');
				greet('Godzilla');

Feel free to play around with this code and get comfortable using a parameter.

You can use more than one parameter

Functions can take any number of parameters.

Type this example that adds two numbers:


				function sum(a,b){
					return a+b;
				}

				console.log( sum(2,3) );
				console.log('The sum is ' + sum(5,4) );

As usual, feel free to play around and write your own functions until you feel comfortable using more than one parameter.

Practical example: Restaurant bill

Try this example that takes a restaurant bill, calculates the tip, and shows it on the page:


					<div id="output"></div>
					<script>
					function tip(bill, percent){
						var result = bill * (percent/100);
						return result;
					}

					var myBill = 25;
					var tipPercent = 20;

					var message = "The bill is $" + myBill
						+ " and the tip is $" + tip(myBill, tipPercent);

					document.getElementById('output').innerHTML = message;
					</script>
