Basic arithmetic

In JavaScript, you can use plus +, minus -, times *, divided by /, and parentheses( ) just like regular math.

Type each of the following examples once and show the results in an HTML div.


				var x = 5 + 3;
				var y = x - 1;
				var m = x * y;
				var d = m / x;
				var whatever = ( (d+1) / 5 ) * 3;

Modular math - Useful for clocks and remainders and stuff

The modulus operator, %, divides two numbers and gives the remainder. For example, 10%7 would give you the remainder of 10/7, which is 3.

Type the following example and change the numbers a couple of times to see the results.


				var x = 5 % 3;
				console.log(x);

Practical examples of modular math

If you split 10 sandwiches among 3 people, how many will be left over? Try this example to find out.


					var leftoverSandwiches = 10 % 3; // get remainder of 10/3

If it's 2:00am now, what time will it be 79 hours from now?


					var time = (2+79) % 24;

Increments and decrements - Useful for keeping count of stuff

Let's say you have a variable, x, that equals 5, and you want to increase it by 1.

Here are three ways to do that:

  1. Set x equal to x+1.

    
						x = x+1;

    If x is 5, this code will change x to 6.

    If x is 6, this code will change x to 7.

  2. This is a short way to increase x by 1.

    
						x += 1;

  3. This is a very short way to increase x by 1.

    
						x++;

    This is convenient, so remember it.

Now that you've seen how increments work, here's a more convenient summary of them:


				var x = 5;

				// these expressions all do the same thing: they add 1
				x = x+1;
				x += 1;
				x++;

				// these expressions all subtract 1
				x = x-1;
				x-=1;
				x--;

				// these expressions both add 5
				x = x + 5;
				x += 5;

Assignment operators: Assign values to variables.

An assignment operator assigns a value to a variable. You've already seen one of them, =, but there are others.

Some of them can make your code shorter.

Type each of the following examples once so you're aware they exist.

(You've already seen some of these in the increments section.)


				var x = 2;
				x += 4; // same as x = x+4;
				x -= 3; // same as x = x-3;
				x *= 10; // same as x = x*10;
				x /= 5; // same as x = x/5;
				x %= 3; // same as x = x%3;
