Boost your skills with assorted projects

Time to test your wings.

This unit has several projects to be done one evening at a time.

There won't be any hand-holding in this time, so it's up to you to create your own solutions.

Good luck!

Getting Started

Copy and paste the following code into a new HTML document: 


				<!DOCTYPE html>
				<html>
				<head>
					<title></title>
					<meta charset="UTF-8">
				</head>
				<body>


				<script>

				</script>
				</body>
				</html>

For convenience, you can start with this HTML document to type the code from the lesson.

Remember to put all scripts between the script tags.

Projects:

  1. Build the front end of a single-page shopping cart.

    People should be able to add or remove items, and the list of items should update along with the total cost.

  2. Validate a form.

    • The form should have the following input fields: name, email, and age.
    • Name should have something in it.
    • Email should have an @ somewhere in it (feel free to do more complicated validation).
    • Age should be a number.
    • There should be helpful error messages.

  3. Create a quiz.

    Your quiz should tally the answers and show a score at the end.

    Be as creative as you want, but make sure it works first.

  4. Create a collapsible navigation menu.

    Click an icon to toggle the menu and make it appear or disappear.

    Feel free to add whatever effects and styles you want.

  5. Create an autocorrecting text box.

    As the user types, the text box should change all instances of "taht" to "that". Feel free to add more autocorrections.

  6. Create a 10-minute countdown clock.

    The clock should show minutes and seconds. Style it however you'd like.

  7. Create a random name picker.

    How it should work: Feed a comma-separated list of names into a text box, click a button, and a random name should be selected from the list.

    Hint: You might have to split a string into an array.

Remember, you don't need to do all of these projects in one sitting.

Just do one at a time, refine it, get comfortable with it, and move on to the next one.

Tip: When in doubt, write it out

Writing an outline of your code in comments can help keep you from getting confused.

Remember: If a project is tricky, you don't want to plan and code at the same time.
