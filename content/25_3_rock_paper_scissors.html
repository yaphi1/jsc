Create a game of rock-paper-scissors

In this unit, we're going to create a rock-paper-scissors game.

The basic rules are as follows:

  • Each player chooses an option: rock, paper, or scissors
  • The winner is determined according to the following rules:
    • Rock beats scissors
    • Scissors beat paper
    • Paper beats rock
  • If both players make the same selection, it's a tie.

A different kind of unit

This section has a twist.

Instead of doing the walkthrough, I'm going to ask you to try to make the game on your own.

Here's what needs to happen:

  • There should be three buttons, one for each option.
  • The opponent should be the computer, and it should pick an option randomly.
  • Each time you click one of the buttons, the game should show your move, your opponents move, and a message telling you whether you won, lost, or tied.

There are many ways to approach this problem, so your path is up to you.

Don't look at the solution until you've gotten your game working.

Solution

If you've created a working game, congrats! If not, keep trying. Only come here if you're finished.

Now copy this code into your text editor and comment on what it is doing at each step:


					<div id="output"></div>
					<div id="buttons">
						<button id="rock">Rock</button>
						<button id="paper">Paper</button>
						<button id="scissors">Scissors</button>
					</div>

					<script>

					var options = ['rock','paper','scissors'];

					var winning = {
						rock: 'scissors',
						scissors: 'paper',
						paper: 'rock'
					};

					var buttons = document.getElementById('buttons').querySelectorAll('button');
					for(var i=0; i<buttons.length; i++){
						buttons[i].onclick = showResults;
					}

					function getOpponentMove(){
						var rand = Math.floor(Math.random()*3);
						return options[rand];
					}

					function showResults(){
						var playerChoice = this.id;
						var opponentChoice = getOpponentMove();

						var output = '';
						
						if(winning[playerChoice]===opponentChoice){
							output = "You win!";
						}
						else if(playerChoice===opponentChoice){
							output = "It's a tie!";
						}
						else{
							output = "You lose!";
						}
						output += "<br>Your choice: "
								+ playerChoice
								+ ", Opponent's choice: "
								+ opponentChoice;

						document.getElementById('output').innerHTML = output;
					}

					</script>

Once you're done commenting, start a new blank document and see if you can create a rock-paper-scissors game from scratch.

You can incorporate what you've learned, and you can add your own improvements too! It doesn't have to match the code example here. It just has to work.
