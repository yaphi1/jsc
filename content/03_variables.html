Getting Started

Copy and paste the following code into a new HTML document: 


				<!DOCTYPE html>
				<html>
				<head>
					<title></title>
					<meta charset="UTF-8">
				</head>
				<body>


				<script>

				</script>
				</body>
				</html>

For convenience, you can start with this HTML document to type the code from the lesson.

Remember to put all scripts between the script tags.

Create a variable

In JavaScript a variable lets you store information.

Try typing the following example:


				<div id="mycontent"></div>

				<script>
				var greeting = 'Hello, my name is Bowser.';
				document.getElementById('mycontent').innerHTML = greeting;
				</script>

This code says we're creating a variable called greeting and we're storing some text in it. Then we're showing the text in a div.

The word var tells the browser that you're creating a variable.

Talk the talk

When someone talks about declaring a variable, that just means creating a variable.

Change a variable

Can you change a variable after it's already set?

Yes!

If you want to change the variable after you've declared it, you don't need the var keyword anymore because you've already created the variable.

Here's how to change a variable (type it and see what happens):


				var greeting = 'original greeting';
				console.log(greeting);

				greeting = 'changed greeting';
				console.log(greeting);

Naming your variables

What can you call your variables?

You can call a variable whatever you want, as long as it fits the following rules:

  • YES, the variable's name can have letters, numbers, underscores, or dollar signs
  • NO, the variable's name CAN'T start with a number
  • NO, the name can't be a word that JavaScript already uses to mean something else (examples you've already seen include "var" or "document")
  • Remember that JavaScript is case-sensitive, so "hellothere" counts as a different variable than "helloThere".

What can a variable actually store?

A variable can store a bunch of stuff, but we'll start with strings and numbers.

Strings

A string is just a piece of text enclosed in single or double quotes (either option is fine).

Numbers

A number is... well... a number like 2 or 126 or -55. Notice that it doesn't get wrapped in quotes.

Examples of strings and numbers: (Type the following code in your document, and then use console.log to show each variable.)


				var string1 = 'hello there';
				var string2 = "hi there";
				var a = 1;
				var b = 2;
				var c = a + b;

Dealing with quotes and apostrophes

What if you want to have a string that contains quotes or apostrophes?

Great question.

As long as the type of quote on the outside doesn't match the type on the inside, you don't have to do anything.

For example, these are fine:


				var greeting = "My name's Bowser.";
				var quote = 'I said, "Hi," and they screamed.';

But if you end up using the same type of quote on the inside and the outside, you'll need to do this:


				var greeting = 'My name\'s Bowser.';

The backslash tells the browser this character is still part of the text string and not part of the code. This technique is called escaping.

Talk the talk: Escaping a character

When someone mentions escaping a character, all that means is putting a backslash in front of the character so that your browser knows the character is not part of the code.

Putting strings together

You can put strings together with a plus sign. That's called concatenating.

Here's an example for you to type out:


				var name = 'Bowser';
				var greeting = 'My name is ' + name;
				var longGreeting = 'Hi there.  My name is ' + name + '. Welcome to my castle.';

				console.log(greeting);
				console.log(longGreeting);

Plus means something different for numbers than for strings

Plus adds numbers, but it connects two strings.

When you add a number and a string, you get a string.

Try out these examples to get a feel for how it all works:


				var a = 2;
				var b = 3;
				console.log(a+b);

				var word = 'hi';
				console.log(word + a);

				var c = '3';
				var d = '4';
				console.log(c+d);

    Hint: Remember to escape certain characters when necessary.

  3. Concatenate two strings and show the result on the page. Repeat 5 times.
  4. Concatenate a string and a variable and show the result on the page. Repeat 5 times.

  5. Use the plus sign to do addition or concatenation on various combinations of strings and numbers.

    Do at least the following combinations:

    • string + string
    • string + variable
    • string + number
    • number + number

    Feel free to play around more.

Congrats! You now know about variables, strings, and numbers.
