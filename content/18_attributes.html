Change an attribute

JavaScript lets you change HTML attributes.

Why is this useful?

One example would be that maybe you want to change a link's destination depending on a user's preferences. Another example would be if you want to style an element directly without class names.

Try changing a link's destination and style as in the example below:


				<a id="mylink" href="old.html">My Link</a>

				<script>
				document.getElementById('mylink').href = 'new.html';

				document.getElementById('mylink').style = 'color:red; background:black;';
				</script>

Remember: Use className to change class

To change the class attribute with JavaScript, you need to use className (because "class" is already reserved for something else in JavaScript).

Example:


					document.getElementById('mydiv').className = 'whatever';

It's so easy to forget this, but please remember className.
