Loops: accomplish a lot with little code

A loop lets you run some code repeatedly without having to write it more than once.

We'll cover two kinds of loops here:

  • while loops
  • for loops

While loop: Repeat a block of code while a certain condition is true

A while loop lets you run some code over and over again while a certain condition is true.

For example, the following script keeps increasing a number by 1 and writing that number in the console while it is less than 10.

Type the following script and run it in your browser to see how it works.


				var x=0;
				while(x<10){
					x++;
					console.log(x);
				}

BEWARE INFINITE LOOPS

An infinite loop happens when a loop never ends.

For example, if you make a loop that counts up from zero and keeps going as long as x is greater than negative five, then it will go on forever.

An infinite loop will crash your browser. Avoid that.

For loop: Repeat a block of code a certain number of times

A for loop lets you run some code a certain number of times.

How a for loop is structured

Here's how a for loop is structured (do NOT type this):


				for(A; B; C){
					// code block you want to repeat
				}

A: Run this statement before the loop starts.

B: Keep repeating the code block while this is true.

C: Run this statement after every time the code block runs.


A real example of a for loop

Type this out:


				for(var x=3; x<20; x=x+2){
					console.log(x);
				}

In this example, the loop says:

A: First create a variable called x and set it to 3. Because we've created it here, x is available only inside this loop.

B: Keep repeating the loop as long as x is less than 20.

C: Each time the loop runs, increase x by 2.

Try changing the conditions of the for loop to get a better understanding of how it works.


A common example of a for loop

Type this example and try it in your browser. It should show numbers from 0 to 9.


				for(var i=0; i<10; i++){
					console.log(i);
				}


Common errors to avoid when using for loops:

  • Infinite loops
  • Forgetting that the for loop conditions need to be separated by semicolons
  • Forgetting that x<10 doesn't include 10

Practical example of a for loop:
Display social media posts

Pretend you're coding the front-end of a social media site. You need to display a list of posts.

For now, you don't need to worry about the content.

Here's how that might look (type this example out and try it in your browser):


				<style>
				.post{
					padding: 20px;
					margin: 20px;
					max-width: 800px;
					background: #eee;
				}
				</style>

				<div id="output"></div>

				<script>
				var content = '';

				for(var i=0; i<10; i++){
					content += '<div class="post">This is post '+i+'</div>';
				}

				document.getElementById('output').innerHTML = content;
				</script>

Explanation:

First this code sets a variable called content.

That variable will store the HTML content you want to put in the output div.

Then the for loop happens like this:

  • Before the loop starts:
    Create a variable called i and set it equal to 0
  • When the loop starts:
    Keep repeating it while i is less than 10
    • When i=0:
      In the content, add a div that says "This is post 0"
      Increase i by 1
    • When i=1:
      In the content, add a div that says "This is post 1"
      Increase i by 1
    • When i=2:
      In the content, add a div that says "This is post 2"
      Increase i by 1
    • ... and on and on (the last one is i=9)

Then we take our newly built content and put it inside our output div.

You can generate any number of posts this way without having to write repetitive code!
