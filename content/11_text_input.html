The basics of text input

To access the text inside an HTML text input box, all you need to do is get the element and put .value on the end.

That's it.

Type the following code to see how it works:


				<input id="mytext" type="text">
				<button id="mybutton">Click me</button>
				<div id="output"></div>

				<script>
				document.getElementById('mybutton').onclick = function(){
					var text = document.getElementById('mytext').value;
					document.getElementById('output').innerHTML = text;
				};
				</script>

If you've done it correctly, you should get a page with a textbox and a button.

If you type in the textbox and click the button, you should see your text displayed on the page. Try typing different things and clicking the button.

Most of that code is stuff you've seen before except for this line:


				var text = document.getElementById('mytext').value;

All it's doing is getting whatever text value is inside the text box and storing it in a variable.

A practical example of text input

Type this code that converts miles to kilometers:


				<input id="miles" type="text"> miles
				<br><button id="convert">Convert to km</button>
				<div id="output"></div>

				<script>
				document.getElementById('convert').onclick = function(){
					var mi = document.getElementById('miles').value;
					var km = mi * 1.609344;
					document.getElementById('output').innerHTML = km + ' km';
				};
				</script>

Now that you have access to text input, you can make your pages much more interactive.
