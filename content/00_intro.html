How to get the most out of this course

First off, remember why you're excited to learn JavaScript

  • Freedom to create what you want
  • Much higher salaries
  • More job options
  • Having more fun with your awesome new skills

These are some of the things you can look forward to when you learn JavaScript.

A lot of people never invest in themselves like this, so by making the decision to learn, you're already way ahead of the people who do nothing and wait for life to happen to them.

Congrats!

Before you begin

You should have some familiarity with HTML and CSS.

If you can do the following things, you know enough:

  • HTML:
    You know how to create a div, a span, a link, a button, and a text input field
  • CSS:
    You know how to style things with an id or a class

If not, take some time to practice those things, and then come back. This course will be here for you when you're ready.

What you'll learn

JavaScript lets you add interactivity to your pages.

Here are some examples of things you can do with JavaScript:

  • Showing and hiding things when users click
  • Validating forms
  • Changing styles dynamically
  • Responding to user actions
  • Interactive games
  • Countdown clocks
  • Calculation helpers
  • Much more stuff than I can list

Tools you'll use

What you'll need:

  • A text editor (I personally prefer Sublime Text, a free download, but feel free to use whatever you want.)
  • A browser

Why those tools?

A lot of courses have their own editors where you learn directly on their site. This is great for convenience, but afterwards, many people say that they don't know how to apply their knowledge in real situations.

This course won't let that happen to you.

In order to get used to coding in real life, it's best for your practice to match real conditions as much as possible.

That means you'll be using your own text editor, and you'll be testing your code in your own browser.

This way, you won't feel lost when the time comes to code in real life because you've been practicing under real conditions all along.

How you'll learn:
The Simple Steps Method

  1. Practical usefulness comes first

    • A lot of courses and books make you slog through a ton of theory before you get to do anything practical, but not this course.
    • This course is structured so that you get to do as many practical, useful things as soon as possible while still getting the theory you need.

  2. Learn small pieces at a time

    • I’d rather tell you one thing and have you remember it than 10 things and have you forget all of them.
    • You'll keep more excitement and momentum this way, and that will make you learn faster.

  3. Practice small pieces of code until they feel automatic and easy

    • You'll actually have to type the code to remember it well. That's important. The concepts in each section build on one another, so 10 extra minutes of practice now can save you 3 hours of confusion later.
    • The course is structured so you'll spend more time actually coding than reading.

    Warning: Don't trick yourself

    Don’t just stop when you understand a concept, as that’s the number 1 way to forget it.

    Keep practicing until it's easy.

    The course is structured so this happens automatically as long as you do the exercises.

  4. Enjoy yourself

    • Each section is a fun new skill you get to try out, so don’t treat it like work.
    • If I ask you to repeat some code, maybe put on some music (and even a timer) and have fun with it.
    • Don't be afraid to learn new things. Every big concept is just a bunch of smaller concepts put together.
