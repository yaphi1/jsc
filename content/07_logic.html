JavaScript Logic

Logic is one of the most fun parts of JavaScript because it lets your code make decisions.

The main building blocks of JavaScript logic are if statements and else statements.

Talk the talk: Conditional statements

In case anyone asks, if and else statements are also called conditional statements. Now you won't be confused when you hear the word "conditional."

To see how conditional statements work, let's look at some examples.

If statement

Type the following code into an HTML document and open it in your browser to see what it does.


				var rating = 8;

				if( rating > 5 ){
					console.log('good rating');
				}

The if statement here says:

  • if rating is greater than 5...
  • then write "good rating" in the console

Since rating is 8, you should see the message.

Try changing rating to 2. The message should disappear.

Else statement

An else statement tells your code what to do when the if statement is false.

Type and run the following code. Then change rating to 2 and see what happens.


				var rating = 8;

				if( rating > 5 ){
					console.log('good rating');
				}
				else{
					console.log('bad rating');
				}

If you've done it correctly, a rating above 5 should show "good rating" and a rating of 5 or below should show "bad rating".

Else if statement

What happens if you want to check many conditions?

You could do this (but do NOT type this one). Try to guess what will happen here.


				var rating = 8;

				// this is true
				if( rating > 7 ){
					console.log('great rating');
				}

				// this is also true
				if( rating > 5 ){
					console.log('good rating');
				}

				else{
					console.log('bad rating');
				}

This code will print "great rating" and "good rating" because both if statements are true. What if we want to exclude one of them?

That's where we use else if.

"Else if" is not some sort of special command. It's just an else statement followed immediately by another if statement.

Type the following example and see what happens. Change rating to 6 and then 4 and see what happens.


				var rating = 8;

				if( rating > 7 ){
					console.log('great rating');
				}
				else if( rating > 5 ){
					console.log('good rating');
				}
				else{
					console.log('bad rating');
				}

Here's what the code does:

  • If the rating is above 7, show "great rating" in the console.
  • If it's not greater than 7, but it's greater than 5, then show "good rating".
  • If none of the previous conditions are true, show "bad rating".

Talk the talk: Control flow

Control flow is just a fancy term for things like if statements or else statements that control the logical flow of your program.

If someone asks if you know about control flow, you can say yes.

Comparison operators

You'll need some way to compare values besides "greater than." Here's a list of the comparisons you can use in JavaScript:


				x>5 // greater than
				x>=5 // greater than or equal to
				x<5 // less than
				x<=5 // less than or equal to

				x==5 // equal to
				x===5 // exactly equal to
				x!=5 // not equal to
				x!==5 // not exactly equal to

What's the difference between equal and exactly equal?

Equal means the values match.

Exactly equal means that the value and the type both match.

For example '5' equals 5. However '5' is not exactly equal to 5 because the first one is a string and the second one is a number.


				// equal
				5==5 // true
				'5'==5 // true

				// exactly equal
				5===5 // true
				'5'===5 // FALSE

Remember: Always use more than one equals sign for comparisons

Remember to use more than one equals sign when comparing for equality.

Forgetting this causes so many errors!

As you have seen in this course, the single equals sign is used to assign a variable. Two or more equals signs are used to check for equality.

I'm going to say it again. Forgetting this causes so... many... errors! Please remember it.
