Logical operators - and, or, and not

To make your logic more powerful, you can use the following logical operators:

  • && (which means "and")
  • || (which means "or")
  • ! (this means "not")

Type the following examples, and see what they do.


				// example of "and"
				var age = 12;
				var height = 58;
				if( age>=10 && height>=42 ){
					console.log('You can go on the rollercoaster.');
				}

				// example of "or"
				var name = 'Bowser';
				if( name=='The Emperor' || name=='Bowser' ){
					console.log('You get the VIP table.');
				}

				// example of "not"
				if( name != 'Gerbil' ){
					console.log('Your name is not Gerbil.');
				}

Note: Remember != means "not equal", and !== means "not exactly equal".

Parentheses

You can use parentheses in your logic too. They work just like in math where they group pieces of the logic together.

Play with the following example to get a feel for how it works.


				var money = 1000000000000;
				var fame = 9001;
				var name = 'Bowser';
				if( (money>999999999 && fame>9000)  || name=='Bowser' ){
					console.log('You can ride the VIP rollercoaster.');
				}

The code above says:

If ( you're super rich and famous ) or your name is Bowser, you can ride the VIP rollercoaster.

Booleans

We've talked about numbers and strings, but now it's time to talk about booleans.

Booleans are just true or false. That's it.

Here's an example. Please type it in once and feel free to play around with it.


				var correctFingerprint = true;

				if( correctFingerprint==true ){
					console.log('correct');
				}

				// this does the same thing as the previous if statement
				// but this is more concise, so do it this way
				if( correctFingerprint ){
					console.log('correct');
				}

Whoa! Why does that last example work?

Here's why:

If statements only check for booleans

Think about it. Every time you've seen an if statement, it always checked if something was true or false.

Here are some arbitrary examples:

  • X is greater than 5? true
  • X is less than 2? false
  • X is greater than 5 and name equals "Bowser"? true

That's right. An if statement can only understand true or false.

An if statement says: "if (thing in parentheses) equals true, then do {the thing in the brackets.}"


					if( ...stuff goes here... ){ // if stuff is true
						// then do this
					}

Shortcut for checking if something is false

Type this out once:


				var youAreBowser = false;
				if( !youAreBowser ){
					console.log("You're not Bowser");
				}

Remember that the exclamation mark means "not". In this case, if you're not Bowser, this code will tell you that.
