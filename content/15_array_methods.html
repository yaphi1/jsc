Push and pop: Add and remove items at the end of an array

The push method adds an item to the end of an array.

The pop method removes an item from the end of an array.

Type the following example to see how push and pop work:


				var names = ['Optimus','Buzz','Elmo'];
				console.log('names before the push: '+names);

				names.push('Kermit');
				console.log('names after the push: '+names);

				names.pop();
				console.log('names after the pop: '+names);

How to remember the difference between push and pop:

The longer word makes the array longer.

Unshift and shift: Add and remove items at the beginning of an array

The unshift method adds an item to the beginning of an array.

The shift method removes an item from the beginning of an array.

Type the following example to see how unshift and shift work:


				var names = ['Optimus','Buzz','Elmo'];
				console.log('names before the push: '+names);

				names.unshift('Kermit');
				console.log('names after the unshift: '+names);

				names.shift();
				console.log('names after the shift: '+names);

How to remember the difference between shift and unshift:

Once again, the longer word makes the array longer.

Tip: pop and shift tell you what they removed from the array

One useful feature of pop and shift is that their returned value tells you what they just removed from the array.

Try the following example to see this in action:


					var deck = ['Ace','Queen','Joker'];
					console.log('deck of cards before the pop: ' + deck);

					var pickedCard = deck.pop();

					console.log('deck of cards after the pop: ' + deck);
					console.log('picked card is: ' + pickedCard);

You don't need to memorize this right away. I just want you to be aware this exists.

Other array methods:

You have more than enough to remember for now, so all you need to know is that other methods exist for working with arrays.

No need to memorize these immediately.

Here are some examples of other array methods:

  • splice - Add or remove items from anywhere in an array
  • join or split - Join the elements of an array into a string or split up a string into an array
  • slice - Grab a section of an array

Here are some good references in case you want to do something with an array but you're not sure how:

  • Mozilla Developer Network (MDN) (Good if you want something detailed and authoritative)
  • W3Schools (Good if you want a quick and readable reference, but if you're not sure about something, double check with MDN)
  • CanIUse.com (Good for checking browser compatibility)

Feel free to try out some of the examples if you're curious.

Talk the talk: A note about references

Years ago, W3Schools had a reputation for having errors. They've generally fixed them, but the reputation stuck.

The reason this matters is that if you cite W3Schools as your number one source for some info, you risk sounding like you've picked up bad habits (based on people's perceptions of the site's earlier reputation).

Personally, I use W3Schools when I already know a concept and I want a quick refresher. I use MDN if a concept is completely new and I want to be sure I know the full story.
