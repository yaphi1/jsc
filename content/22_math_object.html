The Math object - Do more than just basic arithmetic

The Math object is a collection of methods that let you do various math operations besides arithmetic.

Type the following example once to see how it works. There's no need to memorize this; just understand how it works.


				var power = Math.pow(5,2); // 5 to the power of 2
				var roundDown = Math.floor(6.52);
				var roundUp = Math.ceil(2.1);
				var roundToNearest = Math.round(2.4);
				var findLargest = Math.max(2,3,10,5);
				var findSmallest = Math.min(2,3,10,5);

				console.log(power); // try the other values too

Change what's in the console log so you can try different values.

Random numbers

Math.random() gives you a random number between 0 and 1 (including 0, but NOT including 1).

Try the following code a few times to see how it gives different results each time:


				var rand = Math.random();
				console.log(rand);

Now that you have access to randomness, you can manipulate it however you want.

Get a random whole number between 1 and 10

Here are the steps to get a random whole number between 1 and 10:

  1. First, generate a number from 0 to 10 (including zero but NOT including 10).

    
						10*Math.random()

    This code will give you anything from 0 to 9.999999999 etc.

  2. Next, round down to get integers.

    
						Math.floor( 10*Math.random() )

    This code will give you an integer from 0 to 9 (including 9).

  3. Finally, add 1, so you'll get numbers from 1 to 10 instead of 0 to 9.

    
						Math.floor( 10*Math.random() ) + 1

Now you have a random whole number between 1 and 10. Type the following to see it in action:


				var rand =  Math.floor(10*Math.random()) + 1;
				console.log(rand);
